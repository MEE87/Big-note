[[vector property]] ^7347f5
##### 可加性組
0. 相加後仍處於vectror space [^1]
1. 交換律
2. 結合律
3. 0向量存在
4. 反向量存在
##### 標量組
0. 標量乘法後仍在vector space
1. 分配律1 $a(\vec{v_1}+\vec{v_2}) = a\vec{v_1}+a\vec{v_2}$
2. 分配律二(標量相加)
3. 交換率 $(ab)\vec{v} = a(b\vec{v})$
4. 單位標輛存在

[^1]: $\vec{v} + \vec{w} \in V$ for any  vector in V


任何滿足以上式子階可說是vector space

[[判斷向量空間]]
判斷一個子集是否為空集合分三步
1. find non-0 vector
2. closed in addition
3. closed in scalar multiplication

#### [[Linear span]]
A vector space is spanned by multi-vector $\{{v_1,v_2,\dots ,v_m}\}\ m\in \mathbb{R}$
Expression$$V = \alpha_1v_1 + a_2 v_2 + \cdots + a_m v_m \quad m\in \mathbb{R} $$

####  [[linear independence]]
Sometime we will find some vector redundant in linear span. 
We call a subset {$v_1 + v_2 + \cdots + v_m \quad m \in \mathbb{R}$}  when the only method to create 0 vector is when leading coefficient all 0.
